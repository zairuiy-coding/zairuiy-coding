## 🚀 About Me

🎓 I'm **Katherine**, a software engineer focused on **distributed systems, backend infrastructure, and high-performance data pipelines**.  
I enjoy building systems that are **scalable, reliable, and production-ready**.

### 💼 Recent Experience:
- **PayPal – JunoDB (Distributed Caching, 350B+ daily requests)**  
  Executed tech refresh across 3+ pools with 100GB+ data migration achieving zero downtime; managed GKE-based storage platform reducing latency from 2s to <10ms; maintained Java/Spring client library and designed a distributed counter feature for linear write throughput scaling.
- **Tesla (Factory Software Team)**  
  Migrated a legacy .NET ETL system to Golang & React, rebuilding backend as a task scheduler extracting data from 200+ configured sources and publishing to Kafka topics.
- **SciFeCap (Quant Trading)**  
  Built a WebSocket-based real-time data pipeline using the Strategy Design Pattern, reducing database load by 40% while preserving trading-relevant signals.

🔗 Explore my **[Portfolio](https://zairuiy-coding.github.io)** for more details on my projects and contributions.

My focus is on **distributed systems, backend infrastructure, and building scalable production systems**.
